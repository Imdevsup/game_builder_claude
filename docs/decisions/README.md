# Decisions

We use Architecture Decision Records (ADRs).

- One file per decision: `<NNNN>-<short-name>.md`.
- Status: `proposed`, `accepted`, `superseded`.
- Numbering is monotonic. Never reuse a number.

Index:
- [0001-stack-locked.md](./0001-stack-locked.md), initial stack.

Special files:
- [CURRENT_PHASE.md](./CURRENT_PHASE.md), current phase status, blockers, target.
- [CHANGELOG.md](./CHANGELOG.md), append-only log of meaningful changes.
- [OPEN.md](./OPEN.md), open questions parked here until resolved.
