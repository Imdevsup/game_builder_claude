## steps
1. read the phase definition from the bible
2. list each deliverable with an explicit acceptance criterion
3. set CURRENT_PHASE.md to this phase with the entry timestamp
4. open OPEN.md questions specific to this phase

## deliverable
an updated CURRENT_PHASE.md and a phase plan with deliverables.

## house style
- plain english
- no em dashes
- short sentences
- specific over generic
- if blocked, ask, don't invent
